# CleanRoute-AI - Final Project Report

## Project Overview
AI-Powered Waste Management System for Attock city

## Features Implemented
✅ Citizen Complaint Registration with Photo Upload
✅ AI-Powered Waste Detection from Images
✅ Real-Time Map Tracking
✅ Route Optimization with K-Means Clustering
✅ Waste Prediction with Linear Regression
✅ Admin Dashboard for Complaint Management
✅ Tester Dashboard for Field Tasks
✅ Role-Based Access Control (Admin, Citizen, Tester)
✅ Export Reports (PDF/Excel)
✅ Dark/Light Theme Toggle
✅ Mobile Responsive Design

## Technical Stack
- Backend: Django REST Framework
- Frontend: React + Material-UI
- Database: SQLite
- Maps: Leaflet + OpenStreetMap
- ML: scikit-learn
